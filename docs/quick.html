Overview

SIMPLE_CSV_QUICK provides one-liner CSV read/write operations. Supports CSV, TSV, and European-style semicolon-delimited files.

For column-by-name access and RFC 4180 details, use SIMPLE_CSV directly.

Quick Start

local
    csv: SIMPLE_CSV_QUICK
    rows: ARRAYED_LIST [ARRAYED_LIST [STRING]]
do
    create csv.make

    -- Read CSV file to list of rows
    rows := csv.read ("data.csv")

    -- Parse CSV string
    rows := csv.parse ("name,age%NAlice,30%NBob,25")

    -- Write rows to file
    csv.write ("output.csv", rows)

    -- Convert rows to CSV string
    print (csv.to_csv (rows))

    -- Read with headers (returns list of maps)
    across csv.read_with_headers ("data.csv") as rec loop
        print (rec ["name"] + " is " + rec ["age"])
    end

    -- Build rows easily
    rows := csv.rows_from_arrays (<<
        <<"Alice", "30", "NYC">>,
        <<"Bob", "25", "LA">>
    >>)

    -- Write with headers
    csv.write_with_headers ("output.csv", <<"name", "age", "city">>, rows)

    -- TSV (tab-separated)
    csv.use_tabs
    rows := csv.read ("data.tsv")

    -- European CSV (semicolon-separated)
    csv.use_semicolons
    rows := csv.read ("european.csv")
end

API Reference

Reading

FeatureDescription
read (path)Read CSV file to list of rows
read_with_headers (path)Read to list of maps (field name -> value)
parse (csv_string)Parse CSV string to rows

Writing

FeatureDescription
write (path, rows)Write rows to CSV file
write_with_headers (path, headers, rows)Write with header row
to_csv (rows)Convert rows to CSV string

Row Building

FeatureDescription
row (fields)Create row from array of fields
rows_from_arrays (arrays)Create rows from arrays

Configuration

FeatureDescription
set_delimiter (char)Set field delimiter
use_tabsUse tab delimiter (TSV)
use_semicolonsUse semicolon delimiter (European)

Utilities

FeatureDescription
column (rows, index)Extract single column
row_count (rows)Number of rows
column_count (rows)Number of columns
